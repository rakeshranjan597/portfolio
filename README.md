# Rakesh Ranjan — Portfolio

A fast, responsive, single-page portfolio website built with plain HTML, CSS, and JavaScript. No frameworks, no build step.

**Live site:** enabled via GitHub Pages (Settings → Pages → deploy from `main` branch).

## Highlights

- Light/dark theme with system preference detection and persistence
- Scroll-reveal animations and animated stat counters
- Responsive layout with a mobile nav
- Accessible, reduced-motion friendly

## Structure

```
portfolio/
├── index.html      # markup and content
├── styles.css      # design tokens, layout, theming
├── script.js       # theme toggle, nav, reveal, counters
└── rakesh-ranjan-resume.pdf
```

## Run locally

Just open `index.html` in a browser, or serve it:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
